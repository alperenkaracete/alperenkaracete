### 🛡️ Kubernetes & Service Mesh Security Research
Conducting academic research focused on cloud-native security, specifically analyzing and improving security protocols within Kubernetes environments and service meshes.

### 🍓 [Raspberry Pi Yocto Project](https://github.com/alperenkaracete/Raspberry-Yocto-Project)
A full-stack IoT system demonstrating modern microservice communication.
* **Tech Stack:** C++ Backend, REST API, ZeroMQ, React
* **Focus:** Built a C++ backend utilizing ZeroMQ for microservices communication, implemented Modbus monitoring, and developed a REST API.

* **Tech Stack:** C, Socket Programming, GTK
* **Focus:** Low-level network programming, concurrent client handling, and server architecture.

### 🌍 [Tour Agency Database](https://github.com/alperenkaracete/GTU-CSE-Homeworks/tree/main/CSE-414%20Database/Database%20Tour%20Agency%20Project%20-SQL%20Files-Interface-Report)
A comprehensive relational database system managing agencies, agents, tours, and customers.
* **Tech Stack:** MySQL, SQL
* **Focus:** Complex relational database design, query optimization, and data integrity.

### 📚 [Library Management System](https://github.com/alperenkaracete/LibraryManagementSystem)
A complete CRUD application for managing library operations (books, users, librarians).
