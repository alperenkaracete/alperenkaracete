## 🚀 Backend, System Architecture & AI Projects

### 🤖 [OlyKube](https://github.com/alperenkaracete/OlyKube) *(Ongoing)*
A backend platform for orchestrating LLM-based AI agents on Kubernetes, built as part of my MSc thesis research.
* **Tech Stack:** FastAPI, PostgreSQL, Docker, Kubernetes, LangGraph
* **Highlights:**
  * RESTful API with JWT authentication, Redis rate limiting, and structured logging
  * LangGraph ReAct agent with tool-use capabilities and PostgreSQL-persisted conversation history
  * Dynamic agent Pod spawning on Kubernetes with resource-aware scheduling
  * Custom Prometheus metrics for AIOps observability

### 📚 [Library Management System](https://github.com/alperenkaracete/LibraryManagementSystem)
A full-featured CRUD application for managing library operations, written entirely in **Java**.
* **Tech Stack:** Java, MySQL
* **Focus:** Object-Oriented design, clean code principles, layered architecture, and state management across books, users, and librarians.

### 🍓 [Raspberry Pi Yocto Project](https://github.com/alperenkaracete/Raspberry-Yocto-Project)
A full-stack IoT system demonstrating modern microservice communication patterns.
* **Tech Stack:** C++ Backend, REST API, ZeroMQ, React
* **Focus:** C++ backend using ZeroMQ for inter-service messaging, Modbus protocol monitoring, REST API design.

### 💬 [10 Rooms Chat App](https://github.com/alperenkaracete/CSE-472-Chat-Room-Project)
A multi-room chat server built in C using socket programming.
* **Tech Stack:** C, Socket Programming, GTK
* **Focus:** Low-level networking, concurrent client handling, server architecture.

### 🌍 [Tour Agency Database](https://github.com/alperenkaracete/GTU-CSE-Homeworks/tree/main/CSE-414%20Database/Database%20Tour%20Agency%20Project%20-SQL%20Files-Interface-Report)
